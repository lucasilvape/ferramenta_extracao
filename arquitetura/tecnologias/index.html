    📊 Processamento de Dados

    Biblioteca Função Exemplo de Uso
    pandas Manipulação de DataFrames pd.read_sql_query() para queries
    openpyxl Leitura de Excel Upload de arquivos do parceiro
    xlsxwriter Escrita de Excel Exportação de relatórios formatados
    tqdm Barras de progresso Feedback visual em loops longos

    Chunking para Grandes Volumes

    Relatórios com mais de 700.000 linhas são automaticamente divididos em múltiplos arquivos CSV para evitar problemas de memória e limites do Excel.

    🔄 Concorrência

    O projeto utiliza QThreads do PySide6 para executar operações pesadas em background, mantendo a interface responsiva.

    Thread Responsabilidade
    VersaoThread Verifica versão do sistema no banco
    CarregarProdutosConcThread Carrega lista de produtos
    CarregamentoArquivosThread Lê arquivos Excel/CSV do usuário
    RelatorioThread Gera relatórios LoteCap/Recorrência
    ConciliacaoThread Executa pipeline completo de conciliação
    ConsultaThread Busca bilhetes por filtro

    📦 Distribuição

    Ferramenta Função
    PyInstaller Empacota a aplicação em um .exe único
    build.spec Configuração customizada do build

    Dependências Ocultas

    O build.spec inclui hiddenimports para garantir que módulos dinâmicos (como psycopg2 e widgets do PySide6) sejam incluídos no executável.
