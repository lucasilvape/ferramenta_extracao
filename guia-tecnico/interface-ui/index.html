    🎨 Interface (UI)

    A interface gráfica foi desenvolvida utilizando Qt Designer para o layout visual e PySide6 para a lógica.

    📁 Estrutura de Arquivos

    assets/
├── icons/                               # Imagens e ícones (PNG, SVG)
├── qrc_files/
│   └── icons_.qrc                      # Manifesto de recursos Qt
└── ui_files/
    └── main_window_movto_sinistro.ui   ✅ Layout principal ativo

src/ui/
├── __init__.py
├── icons_rc.py                         # Recursos compilados (gerado)
└── main_window_movto_sinistro.py       ✅ UI compilada — arquivo oficial (gerado)

    🔄 Fluxo de Conversão

    O Qt Designer gera arquivos .ui (XML) que precisam ser convertidos para Python.

    flowchart LR
    A[Qt Designer] -->|Salva| B[.ui file<br>XML]
    B -->|pyside6-uic| C[.py file<br>Classe Python]

    D[Ícones/Imagens] -->|Organiza| E[.qrc file<br>Manifesto]
    E -->|pyside6-rcc| F[_rc.py file<br>Recursos binários]

    C --> G[MainWindow]
    F --> G

    🛠️ Comandos de Conversão

    Execute na raiz do projeto

    Todos os comandos assumem que você está na pasta raiz do projeto.

    Converter Layouts (.ui → .py)

    Converter layout principal
    pyside6-uic assets\ui_files\main_window_movto_sinistro.ui -o src\ui\main_window_movto_sinistro.py

    Converter Recursos (.qrc → .py)

    Converter ícones
    pyside6-rcc assets\qrc_files\icons_.qrc -o src\ui\icons_rc.py

    Classe Ui_MainWindow

    Gerada automaticamente pelo pyside6-uic. Contém:

    Classe MainWindow

    Implementada manualmente em src/main1.py. Herda da UI gerada e adiciona lógica.

    src/main1.py
    from ui.main_window_movto_sinistro import Ui_MainWindow

from pages.financeiro import Financeiro
from pages.conciliacao import Conciliacao
from pages.consulta_apolice import ConsultaApolice
from pages.movto_sinistro import Sinistro

class MainWindow(QMainWindow, Ui_MainWindow):
    def __init__(self):
        super(MainWindow, self).__init__()
        self.setupUi(self)  # Carrega a UI gerada

        self.setWindowTitle("Kovr - Sistema de geração de relatórios")
        self.setWindowIcon(QIcon(":/icons/icons/logo_kovr2.png"))
        self.showMaximized()

        # Instancia os controladores das páginas
        self.financeiro = Financeiro(self)
        self.conciliacao = Conciliacao(self)
        self.consulta_apolice = ConsultaApolice(self)
        self.sinistro = Sinistro(self)

    🎭 Menu Lateral Animado

    O menu lateral possui animação de expansão/contração.

    src/main.py
    def leftMenu(self):
    width = self.menu_esquerdo.width()
    newWidth = 200 if width == 9 else 9

    self.animation = QPropertyAnimation(self.menu_esquerdo, b"maximumWidth")
    self.animation.setDuration(500)
    self.animation.setStartValue(width)
    self.animation.setEndValue(newWidth)
    self.animation.setEasingCurve(QEasingCurve.InOutQuart)
    self.animation.start()

    🔐 Controle de Permissões na UI

    Os botões de navegação são habilitados/desabilitados conforme as permissões do usuário.

    src/main.py
    def verificar_e_carregar_permissoes(self):
    usuario_atual = getpass.getuser()

    paginas_interface = {
        'home': self.btn_home,
        'financeiro': self.btn_financeiro,
        'conciliacao': self.btn_conciliacao,
        'consulta_item': self.btn_consulta_item,
        'movto_sinistro': self.btn_movto_sinistro,
        'contatos': self.btn_contatos,
        'sobre': self.btn_sobre
    }

    permissoes = database.obter_permissoes_usuario(usuario_atual)

    for nome_pagina, botao in paginas_interface.items():
        estado = permissoes.get(nome_pagina, False)
        botao.setEnabled(estado)  # True = habilitado, False = desabilitado

    Sincronização com o banco

    O dicionário paginas_interface deve ter exatamente as mesmas chaves registradas na tabela de permissões do banco de dados. Se uma nova página for adicionada à UI, inclua-a aqui e cadastre-a no banco. Consulte Banco de Dados para detalhes sobre a tabela de permissões.

    🖼️ SplashScreen

    Uma tela de carregamento (QSplashScreen) é exibida enquanto a MainWindow inicializa.

    src/main1.py
    if __name__ == "__main__":
    app = QApplication(sys.argv)

    splash_pix = QPixmap(":/icons/icons/logo_kovr2.png")
    splash = QSplashScreen(splash_pix, Qt.WindowStaysOnTopHint)
    splash.show()
    app.processEvents()

    window = MainWindow()
    window.show()
    splash.finish(window)

    sys.exit(app.exec())

    Boas práticas

    A SplashScreen melhora a percepção de desempenho ao usuário enquanto os controladores de página, conexões de BD e threads são inicializados.

    ⚠️ Importante

    Não edite arquivos gerados manualmente

    Os arquivos main_window_movto_sinistro.py e icons_rc.py são gerados automaticamente. Qualquer alteração manual será perdida na próxima conversão com pyside6-uic.

    Onde fazer alterações:

    • Layout visual → Qt Designer (.ui)
    • Lógica de negócio → src/main1.py ou controladores em src/pages/
